Как использовать

1. GitHub и Telegram
Эта страница подходит для открытия по HTTPS из публичного репозитория и как Mini App через меню Telegram-бота.
2. Google Sheets
Apps Script файлы лежат в репозитории. После деплоя Web App вставьте URL в поле выше, и база начнёт синхронизироваться.
3. Безопасность
URL Apps Script лучше не публиковать открыто. В этой версии он хранится локально на каждом устройстве.
